# CatVTON Setup Guide

## Recommended approach

Use CatVTON in inference mode only.
Do not train the model from scratch.

Because the current device is a MacBook Air, the recommended setup path is:
- Google Colab GPU, or
- another cloud GPU environment

## Setup stages

1. prepare test images
2. get one manual CatVTON run working
3. save one output image successfully
4. document input and output
5. then connect to project wrapper script

## First success condition

The first milestone is not full integration.
The first milestone is:
- one renter image
- one supported garment image
- one generated output image
